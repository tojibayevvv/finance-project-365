# 💰 Finance Manager (Telegram Bot + Dashboard)

## 📌 Overview
This project is built for small and medium businesses in Uzbekistan to manage their finances in one place. It allows users to log income and expenses via a Telegram bot using text or voice messages, and monitor everything through a web dashboard.

---

## 🚀 Features

### 🤖 Telegram Bot
- Log transactions via text or voice
- Automatic parsing of amount, type, and category
- Smart responses and confirmations
- Query financial data (e.g. "How much did we spend this week?")
- Fallback transcription for reliability

### 🌐 Web Dashboard
- Overview page with income, expenses, and net
- Transactions page with:
  - Filtering
  - Edit & delete
  - Excel export
- Analytics page with charts
- Collapsible modern sidebar UI
- Quick-add transaction form

---

## 🛠 Tech Stack
- Backend: Node.js, Express
- Database: Supabase (PostgreSQL)
- Frontend: React (Vite), styled-components
- Bot: node-telegram-bot-api
- Charts: Recharts
